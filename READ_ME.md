Política de Privacidad

Última actualización: 9 de marzo de 2026

1. Introducción

Esta política de privacidad describe cómo la aplicación Gasto Diario (en adelante, "la aplicación") maneja la información del usuario.

La aplicación está diseñada para ayudar a los usuarios a registrar y visualizar sus gastos personales de forma simple y rápida.

2. Información que recopila la aplicación

La aplicación permite registrar información relacionada con gastos personales, incluyendo:

Cantidad del gasto

Categoría del gasto

Fecha del gasto

Nota opcional

Esta información es ingresada manualmente por el usuario con el propósito de organizar sus finanzas personales.

3. Almacenamiento de la información

Toda la información registrada en la aplicación:

Se almacena únicamente en el dispositivo del usuario

No se envía a servidores externos

No se comparte con terceros

No se sincroniza con servicios en la nube

La aplicación utiliza almacenamiento local del dispositivo para guardar los datos de gastos.

4. Uso de la información

La información registrada se utiliza exclusivamente dentro de la aplicación para:

Mostrar la lista de gastos registrados

Permitir crear, editar y eliminar gastos

Generar estadísticas y visualizaciones, como:

totales de gastos

gráficos de categorías

tendencias de gastos

Esta información nunca se utiliza con fines comerciales ni publicitarios.

5. Compartición de información

La aplicación no comparte información personal ni datos del usuario con terceros.

No se recopilan datos para:

publicidad

análisis externos

seguimiento del usuario

marketing

6. Permisos del dispositivo

La aplicación no solicita permisos sensibles del dispositivo como:

ubicación

contactos

cámara

micrófono

7. Seguridad

Dado que todos los datos se almacenan localmente en el dispositivo del usuario, el control y protección de esa información dependen de las medidas de seguridad del propio dispositivo.

8. Privacidad de menores

La aplicación no está dirigida específicamente a menores de 13 años y no recopila intencionalmente información personal de menores.

9. Cambios a esta política

Esta política de privacidad puede actualizarse ocasionalmente para reflejar cambios en la aplicación o en los requisitos legales.

Se recomienda revisar esta página periódicamente para mantenerse informado sobre cualquier cambio.
